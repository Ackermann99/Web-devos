# Simon Game


## Game Information

### The game has same rules as the Simon Game.
### User needs to press any keyboard key to start the game.
### There are four color buttons (Green,Red,Yellow,Blue) each having different sound sound clip (all Sound sub folder).
### First a random sequence is generated and user has to follow the pattern, the sequence is always increased by one step with each increasing level.
### If the user fails to keep up with the sequence, its game over. User then can re-start the game with pressing any keyboard key.
